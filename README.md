# BMW-Sales-Analysis-2010-2024-
BMW Sales Analysis
Overview

This project focuses on analyzing BMW sales data to uncover business insights related to sales performance, product trends, and customer preferences. The analysis is performed using a complete data analytics workflow including data cleaning, exploration, querying, and dashboard visualization.

Objectives
Analyze overall sales performance and revenue trends
Identify top-performing regions and car models
Understand customer preferences (fuel type, transmission, color)
Evaluate the impact of pricing on sales


Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (MySQL) for business queries
Power BI for interactive dashboard
Jupyter Notebook for EDA


Workflow
Data Loading & Cleaning (Python)
Handled missing values and duplicates
Converted data types
Created new feature: Revenue
Exploratory Data Analysis (EDA)
Analyzed distributions and trends
Identified key patterns in sales and pricing


SQL Analysis
Wrote queries to answer business questions
Aggregated data by region, model, and year
Dashboard Creation (Power BI)
Built two reports:
Sales Performance Analysis
Product & Customer Insights
Designed interactive visuals and KPIs


Dashboard Highlights:
Sales Performance Analysis
Total Sales & Revenue
Sales Trend Over Time

Product & Customer Insights
Top Models
Fuel Type & Transmission Preferences
Price vs Sales Analysis

Key Insights
Sales show consistent growth over time
Certain regions dominate overall sales
Mid-range cars are most preferred by customers
Automatic transmission is more popular
High-priced cars generate more revenue but lower sales volume

How to Run
Load dataset in Python
Perform EDA and cleaning
Upload data to MySQL
Run SQL queries
Connect Power BI to database and build dashboard

Project Value

This project demonstrates end-to-end data analytics skills including data cleaning, analysis, SQL querying, and dashboard visualization, making it suitable for entry-level data analyst roles.
